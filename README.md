## Getting Started

### Prerequisites

Nodejs with npm - lastest version.

```
Nodejs 12.6.0
```

### Installing

Download or clone the repository, then

```
npm install
```

And start open the projects for developing. It will go to http://localhost:9000

```
npm run sass
```
